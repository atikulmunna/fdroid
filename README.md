# NativeNote F-Droid repository

Official F-Droid repo for **NativeNote** — private, offline notes with real
on-device AI.

## Install

Add this URL in the F-Droid client (Settings → Repositories → +):

```
https://atikulmunna.github.io/fdroid/repo?fingerprint=35B023550586C174E64ECDEB8AED1D63BCA0D73D2B714F60AF7E454787B6FD89
```

Then search for **NativeNote** and install. Updates arrive automatically.

> from the app's signed APKs by `fdroid update`. The repo signing key and its
